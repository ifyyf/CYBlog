# vue+spring博客园系统

## 介绍
多人博客园项目，前端vue，后端spring，数据库mysql，用的jdbcTemplate连接，第一次接触spring框架写的项目，由于是初学，复用性肯定是没有考虑的，放上来只是为了纪念一下第一次后端开发

## 复用性很低，因为用的jdbcTemplate进行的连接，基本上不太好重构
